# KeepRule Investment Scorecard

Rate any stock using the same criteria legendary investors like Warren Buffett use. Get an instant score and see how your potential investment measures up.

## Try It

**[Use the Scorecard](https://henu-wang.github.io/keeprule-investment-scorecard/)**

## Features

- Score stocks across multiple dimensions (moat, management, valuation, etc.)
- Based on principles from Buffett, Munger, and Graham
- Visual score breakdown
- Mobile-friendly design
